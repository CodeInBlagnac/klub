Utilisation de Meteor
====

Installer meteor:

`$> curl https://install.meteor.com/ | /bin/sh`

Créer une application :

`$ meteor create monapp`

Le lancer en local :

`$ cd monapp`

`$ meteor`

=> Meteor server running on: http://localhost:3000/

Pour déployer :

`$> meteor deploy monapp.meteor.com`
